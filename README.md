# Resonance

Real-time guitar tuner in the browser (React + TypeScript + Vite). Uses the Web Audio API `AnalyserNode` FFT for pitch detection.

## Run

```bash
npm install
npm run dev
```

Open the URL Vite prints. Use **localhost** or **HTTPS** so the microphone works.

```bash
npm run build
```

Outputs to `dist/`.
